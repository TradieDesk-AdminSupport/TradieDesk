# TradeAdmin - Outsourced Admin Support for Tradies

A professional website for outsourced admin support services specifically designed for tradies businesses.

## Features

✅ **Responsive Design** - Works seamlessly on mobile, tablet, and desktop  
✅ **Professional Styling** - Modern blue and gold color scheme  
✅ **Key Sections** - Home, Services, About, Contact  
✅ **Contact Form** - With validation and user feedback  
✅ **Smooth Navigation** - Animated scrolling and navigation  
✅ **Mobile Friendly** - Fully optimized for all screen sizes  

## Services Offered

- 📋 Invoice Management
- 📊 Bookkeeping
- 📧 Email & Calendar Management
- 📞 Customer Communication
- 📝 Job Quoting
- 📱 Administrative Support

## Getting Started

1. Clone this repository
2. Open `index.html` in your web browser
3. No build process or dependencies required!

## Customization

### Change the Company Name
Replace "TradeAdmin" throughout the files with your business name.

### Update Contact Information
Modify the contact form in `index.html` to point to your email service.

### Change Colors
Edit the color values in `css/styles.css`:
- Primary Blue: `#1e3a8a`
- Accent Gold: `#fbbf24`
